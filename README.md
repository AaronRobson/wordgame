# wordgame
A solver for wordgames such as scrabble and countdown

## Install locally
```bash
python3 -m pip install . --user
```

## Run program
```bash
python3 -m wordgame abcd --game countdown
```
Which may result in words such as `bad`.

## Run unit tests
```bash
python3 -m unittest
```
or
```bash
python3 setup.py test
```
